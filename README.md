# README - Hướng dẫn cài đặt VSPE

## 1. Cài đặt

* Tải và cài VSPE
* Mở bằng quyền **Administrator**

---

## 2. Tạo cổng COM ảo (Pair)

* Vào **Device → Create new → Pair**
* Nhập 2 cổng (VD: COM3 ↔ COM4)
* Nhấn **Finish**

---

## 3. Chạy cấu hình

* Nhấn **Start (Play)**
* Nếu lỗi → kiểm tra cổng có bị trùng hoặc đang bị chiếm không

---

## 4. Sử dụng

* Kết nối 2 ứng dụng:

  * App A → COM3
  * App B → COM4
* Dữ liệu sẽ truyền qua lại giữa 2 cổng

---

## 5. Lưu cấu hình

* **File → Save** để lưu (.vspe)
* Dùng lại bằng **File → Open**

---
